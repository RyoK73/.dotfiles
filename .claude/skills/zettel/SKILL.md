# 永続ノート作成（Zettelkasten）

`$ARGUMENTS` のテーマで永続ノートを作成する。

## 手順

### 1. テーマ確認
引数が空なら「永続ノートのテーマを引数として渡してください」と伝えて終了。

### 2. 既存ノートの横断検索
`~/dev/LIFE/zettelkasten/` 全体をgrepで検索し、関連する既存ノートを特定する：

```bash
grep -rl "<キーワード1>" ~/dev/LIFE/zettelkasten/
grep -rl "<キーワード2>" ~/dev/LIFE/zettelkasten/
```

関連ノートが見つかったらタイトルと概要を提示する。

### 3. 対話による内容の言語化

以下の質問を通じて内容を共同構築する：

**Q1**: 「このテーマについて、あなたが最も大切だと思う洞察を1文で言うとすれば？」

**Q2**: 「その洞察に至った経緯・理由は？」

**Q3**: 「これは既存の〇〇ノートとどう関連していますか？」（関連ノートがあれば）

**Q4**: 「このノートに適切なタグを2〜3個挙げてください」

収集した回答を整理して下書きを提示し、「このような内容でよろしいですか？」と確認する。

### 4. 永続ノートの作成
`~/dev/LIFE/zettelkasten/permanent/YYYY-MM-DD-<slug>.md` を作成：

```markdown
---
id: YYYYMMDDNNN
title: <タイトル>
date: YYYY-MM-DD
type: permanent
tags: [<タグ>]
links: [<関連ノートのid>]
---

<本文>

## 関連ノート

- [[<関連ノートid>]] — <タイトル>
```

### 5. 双方向リンクの設定
関連ノートのフロントマター `links:` にこのノートのidを追記する。

### 6. 報告
「🧠 永続ノートを作成いたした: `<ファイルパス>`」と報告する。
