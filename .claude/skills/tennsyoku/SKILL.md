# 転職活動ダッシュボード

$CURRENT_DATE 時点の転職活動全体をまとめて確認する。

## 手順

### 1. データ収集（並列で実行）

以下を同時に読み込む：

1. `~/dev/LIFE/career/applications.md` — 応募・選考状況
2. `~/dev/LIFE/career/curriculum.md` — 学習カリキュラム進捗
3. `~/dev/LIFE/career/README.md` — 全体のToDo状況
4. `~/dev/LIFE/career/companies/` 配下の全 `.md` ファイル（`_template.md` を除く）

### 2. ダッシュボードを出力

以下の形式で出力する。空データは「(なし)」と表示し、エラーにしないこと。

```
# 転職活動ダッシュボード（YYYY-MM-DD）

## 選考状況
- 気になる企業:     N社
- 書類応募済み:     N社
- 書類選考中:       N社
- 面接調整・進行中: N社
- 最終面接以降:     N社
- 完了（内定/不合格）: N社

アクティブな選考:
  - 企業名 ▶ 現在ステージ（次のアクション）
  ...

## 学習進捗
  - Next.js:    ██████░░░░ 60%
  - TypeScript: ████░░░░░░ 40%
  - React:      ████░░░░░░ 40%
  - ...（curriculum.md の全スキルを表示）

## 企業リサーチ済み
  - 企業名（志望度 ★★★☆☆）
  ...

## 今週やるべきこと TOP3（提案）
1. ...（最も緊急・重要なアクション）
2. ...
3. ...
```

#### 進捗バーのルール

進捗%を10段階の棒グラフで表現する。例:
- 0%  → `░░░░░░░░░░`
- 30% → `███░░░░░░░`
- 100%→ `██████████`
