# 企業リサーチ & ノート作成

`$ARGUMENTS` で指定した企業を調査し、`career/companies/` にプロファイルを作成する。

## 手順

### 1. 引数の確認

引数（`$ARGUMENTS`）が空の場合：
- `~/dev/LIFE/career/companies/` 配下の `.md` ファイル（`_template.md` を除く）を一覧表示
- 「どの企業をリサーチいたしますか？企業名を教えてください」と尋ねて終了

引数がある場合は企業名として扱い、次のステップへ進む。

### 2. 既存ファイルの確認

`~/dev/LIFE/career/companies/` 配下を確認し、同名（またはスラッグが近い）ファイルが存在するか確認する。
存在する場合は「既存のリサーチファイルが見つかりましたが、上書き更新いたしますか？」と確認する。

### 3. WebSearch による情報収集

以下のクエリで並列検索する：

1. `"{企業名}" 採用 エンジニア 求人`
2. `"{企業名}" 技術スタック OR テックブログ OR 開発環境`
3. `"{企業名}" 会社概要 OR サービス OR SaaS 評判 働き方`

収集する情報：
- 企業の事業内容・プロダクト概要
- 技術スタック（フロント・バック・インフラ）
- 従業員数・設立年・本社所在地
- リモートワーク状況・年収レンジ
- 企業文化・開発スタイル

### 4. 企業プロファイルの作成

スラッグは企業名をローマ字 or 英語に変換した小文字（例: `株式会社サイボウズ` → `cybozu`）。

`~/dev/LIFE/career/companies/{スラッグ}.md` を作成する：

```markdown
# {企業名}

## 基本情報

| 項目 | 内容 |
|------|------|
| 企業名 | {企業名} |
| 業種 | {業種} |
| 従業員数 | {従業員数} |
| 設立 | {設立年} |
| 本社 | {所在地} |
| 採用形態 | 正社員 / 業務委託 / その他 |
| 求人URL | {URL（見つかった場合）} |
| 調査日 | {$CURRENT_DATE} |

## プロダクト・事業

{事業内容・プロダクトの説明}

## 技術スタック

- フロントエンド：{技術名}
- バックエンド：{技術名}
- インフラ：{技術名}
- その他：{技術名}

## 働き方

| 項目 | 内容 |
|------|------|
| リモート | {フルリモート / ハイブリッド / 出社 / 不明} |
| 勤務時間 | {勤務時間・フレックス等} |
| 年収レンジ | {年収レンジ（見つかった場合）} |
| 入社時期 | {随時 / 要相談} |

## 志望度・メモ

**志望度：** ★☆☆☆☆（調査後に更新）

{なぜ気になったか、懸念点、調べたいこと}

## 選考状況

| 日付 | ステップ | メモ |
|------|----------|------|
| | | |
```

### 5. Zettelkasten 文献ノートの作成

`~/dev/LIFE/zettelkasten/literature/` 配下に文献ノートを作成する：

ファイル名: `YYYY-MM-DD-company-{スラッグ}.md`

```markdown
---
id: YYYYMMDDNNN
title: 企業リサーチ: {企業名}
date: {$CURRENT_DATE}
type: literature
tags: [転職, 企業リサーチ, {業種}]
links: []
sources: [{調査したURL}]
---

## 概要

{企業の一言サマリ}

## 主要なポイント

- {技術的な特徴}
- {文化・働き方の特徴}
- {志望度の根拠}

## 関連する洞察

→ 詳細プロファイル: `career/companies/{スラッグ}.md`
```

### 6. 報告

以下を出力する：

```
📋 リサーチ完了でござる

企業名: {企業名}
プロファイル: ~/dev/LIFE/career/companies/{スラッグ}.md
文献ノート: ~/dev/LIFE/zettelkasten/literature/YYYY-MM-DD-company-{スラッグ}.md

【ひとことサマリ】
{企業の魅力と技術的な特徴を2〜3文で}

志望度の暫定評価: ★☆☆☆☆（/job-research のプロファイルで更新してください）
次のアクション: /job-track {企業名} apply で応募登録できまする
```
